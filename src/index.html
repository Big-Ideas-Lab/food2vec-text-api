food2vec Text Corpus API

GET /recipe

Description: get text corpus of one search query

Parameters:

query* (string) - The string/word to search recipies for. Example: localhost:5000/recipe?query=banana

number (int) - The number limit of recipies to return from Spoonacular. Example: localhost:5000/recipe?query=banana&number=5

GET /recipes

Description: get text corpus of multiple search queries.

Parameters:

queries* (array of strings) - The words to search recipies for. Example: localhost:5000/recipes?queries=banana,apple,pie

number (int) - The number limit of recipies to return from Spoonacular. Example: localhost:5000/recipe?queries=banana,apple,pie&number=5

* required
